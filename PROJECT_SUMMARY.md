# Mu Blog 项目总结

## 项目概述

Mu Blog 是一个现代化的博客系统，专为EdgeOne边缘计算平台设计和优化。该项目采用了Vue.js全栈技术栈，集成了Turso边缘数据库和Twikoo评论系统，提供了完整的博客管理和展示功能。

## 技术架构

### 前端技术栈
- **Vue 3**: 渐进式JavaScript框架，使用Composition API
- **Vue Router**: 客户端路由管理
- **Pinia**: 状态管理解决方案
- **Vite**: 现代化构建工具
- **Axios**: HTTP客户端
- **Marked**: Markdown渲染引擎
- **Highlight.js**: 代码语法高亮

### 后端技术栈
- **Node.js**: JavaScript运行时环境
- **Express**: Web应用框架
- **JWT**: JSON Web Token身份认证
- **Bcrypt**: 密码加密
- **Turso(libSQL)**: 边缘数据库

### 部署平台
- **EdgeOne**: 边缘计算平台
- **自动扩缩容**: 根据流量动态调整资源

## 使用说明

### 快速开始
1. 克隆项目代码
2. 安装依赖：`npm install`
3. 配置环境变量
4. 启动开发服务器：`npm run dev`
5. 访问 http://localhost:3000

### 生产部署
1. 配置EdgeOne环境
2. 设置Turso数据库
3. 配置环境变量
4. 运行部署脚本：`deploy-edgeone.bat`
5. 在控制台验证部署结果
